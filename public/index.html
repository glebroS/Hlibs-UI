Sampling ›
0.10
What is Min-P?

Filters out tokens below a fraction of the top token's probability. 0.05–0.15 = creative & coherent. 0.0 = off. Preferred over Top-P for modern models.

0.30
What is Temperature?

0.1–0.4 = precise, logical, coding tasks. 0.6–0.9 = balanced conversation. 1.0+ = creative writing, brainstorming.

1.10
What is Rep. Penalty?

Discourages the model from repeating itself. 1.0 = off. 1.1 = light nudge (recommended). 1.3+ = aggressive, may hurt quality.

40
What is Top-K?

Limits sampling to the top K most likely tokens at each step. 0 = disabled. 20–50 = tight & coherent. 100+ = more variety. Works alongside Min-P.

Context & Output ›
8192
What is num_ctx?

Total tokens the model can "see" at once (prompt + response). 4096–8192 = most tasks. 16k–32k = long docs/code. Higher = more VRAM used.

2048
What is num_predict?

Hard cap on response length. 512 = short answers. 2048 = standard. 4096+ = long code or essays. Use -1 for unlimited (set slider max).

Advanced ›
OFF
What is Mirostat?

Adaptive sampling algorithm that maintains a target perplexity (τ). OFF = use Min-P+Temp. v2 = best for long coherent generations. Overrides Top-K/Min-P when on.

5.0
What is Tau?

Target perplexity. 3–5 = focused. 7–10 = creative. Only active when Mirostat is ON.

Random
What is Seed?

Fix a seed to get reproducible outputs for the same prompt+settings. Leave blank for random. Useful for debugging or exact comparisons.

Reasoning & Persona ›
What does this do?

Injects a system instruction telling the model to wrap its internal reasoning in <think> tags before answering. Reveals the model's "chain of thought" in a collapsible panel.

What is a system prompt?

Sets the model's persona and behavior for the entire conversation. Takes priority over Force Think if both are set. Leave blank for default behavior.
